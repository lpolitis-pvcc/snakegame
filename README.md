# Snake Game (using PyGame)

A snake game made using Python created by Leo Politis. 

Design Document: https://docs.google.com/document/d/1hz1gHkmcdFHp9RkS6ksEjKN1sZSBgNen/edit?usp=sharing&ouid=110340679723330585869&rtpof=true&sd=true 

To play, you must first install Python and PyGame. 
To do this, you have to go to Python.org and download Python.
Next, install PyGame (use https://www.pygame.org/wiki/GettingStarted)
